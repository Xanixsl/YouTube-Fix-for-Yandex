# 📺 YouTube Fix for Yandex

![Banner](https://github.com/user-attachments/assets/05972b03-8bcd-4eab-a992-6377db5e68ff)

**A powerful userscript that eliminates YouTube bugs, restores broken features, and gives you full control over the interface — with deep optimisations specifically for Yandex Browser.**

After installation a **⚙️ gear button** appears in the YouTube top bar — right next to the **+Create** button and your profile icon.  
Click it to open the full settings panel — no browser extensions, no popups, just a floating window built into the page itself.

## 📥 Installation

1. Install a userscript manager:

   [![Violentmonkey](https://img.shields.io/badge/Violentmonkey-recommended-8957e5?style=for-the-badge&logo=googlechrome&logoColor=white&labelColor=0d1117)](https://violentmonkey.github.io/get-it/)&nbsp;
   [![Tampermonkey](https://img.shields.io/badge/Tampermonkey-install-f0b429?style=for-the-badge&logo=googlechrome&logoColor=white&labelColor=0d1117)](https://www.tampermonkey.net/)

2. Click to install the script:

   [![Install latest version](https://img.shields.io/badge/⬇️_Install-Latest_Version-238636?style=for-the-badge&logo=javascript&logoColor=white&labelColor=0d1117)](https://raw.githubusercontent.com/Xanixsl/YouTube-Fix-for-Yandex/main/youtube-fix-yandex.user.js)

3. Refresh YouTube — the settings panel will appear on the page.

> The script auto-updates when new versions are released.

## ⚙️ Features

The settings panel has **3 tabs**:

### 🗂 Tab 1 — General

**Interface**

| Feature | Description |
|---------|-------------|
| Hide chips (filters) | Hides the filter bar on the home page |
| Chipbar background height | Adjusts the frosted-glass strip height (px) |
| Hide chipbar background | Completely hides the masthead background strip |
| Compact mode | Reduces spacing between videos for denser layout |
| Hide Shorts | Removes Shorts section and all short video recommendations |
| Hide "More topics" | Removes topic shelves from the home page |
| Hide slowdown warning | Removes RF-region slowdown notification |
| Hide empty blocks | Hides broken placeholder blocks in the feed |
| Fix channel card | Fixes the sliding channel card on all channel tabs |
| Restore chips on Videos tab | Ensures sorting chips are always visible on channel pages |
| Playlists on channels | Restores playlists to channel pages |
| Cinema mode on watch pages | Auto-enters Cinema Mode when opening a video: centers the player on a dark background, hides header, sidebar and navigation. Configure via the sliders (≡) button in player controls; exit via the camera button |

**Bug Fixes**

| Feature | Description |
|---------|-------------|
| Auto-dismiss "Video paused" | Auto-clicks continue when YouTube pauses on idle |
| Fix dark theme flash | Prevents white flash during page navigation in dark theme |
| Fix search results grid | Corrects video grid layout on search results page |
| Fix mini-player overlay | Fixes z-index issues with the YouTube mini-player |
| Smooth scroll optimization | Reduces scroll stuttering on feed pages |
| Fix sidebar rendering | Fixes sidebar glitches during navigation |
| Force H264 codec | Disables VP9/AV1 codecs to fix video stuttering and freezing |

**Language** — Auto-detect / English / Russian

---

### 🗂 Tab 2 — Yandex Fixes *(Yandex Browser only)*

**Yandex Browser Fixes**

| Feature | Description |
|---------|-------------|
| SPA navigation fix | Fixes Back button and page transition issues |
| Page overflow fix | Eliminates double-scroll caused by Yandex optimizations |
| Fullscreen mode fix | Removes UI artifacts in fullscreen video |
| Player controls fix | Fixes player control rendering in Yandex Browser |

**Video Grid Settings** — videos per row, chipbar vertical offset, video block offset

**Experimental** — grid fix (3-per-row bug), performance mode, alternative shift fix

---

### 🗂 Tab 3 — Settings

**Appearance** — thumbnail size (4 options)

**Window** — position (5 options), font size, width, corner radius, smooth animation toggle

**Color Scheme** — 13 built-in themes:

`YouTube` · `Improved` · `Midnight` · `Sunset` · `Ocean` · `Emerald` · `Rose` · `Dark Pink` · `Frost` · `Sky` · `Classic` · `Aurora` · `Custom`

**Custom Colors** — manually override any individual UI color variable

---

### 🎨 Style Editor

A full built-in visual editor with 4 tabs:

| Tab | Contents |
|-----|----------|
| **Presets** | Save, load, delete, export/import themes as JSON |
| **Colors** | Visual color scheme builder with 10+ built-in themes |
| **Background** | Background image (URL, opacity, blur, size) for the settings window or YouTube page |
| **Custom CSS** | Write and apply custom CSS rules directly to YouTube |

---

### 🎬 Cinema Mode

Enabled via the **Cinema mode on watch pages** toggle in the **General** tab, Interface section. When active, automatically enters Cinema Mode on every watch page — centering the player on a dark background, hiding the header, sidebar and navigation.

**In-player quick-settings panel** — click the sliders button (≡) in the top-right player controls:

| Parameter | Description |
|-----------|-------------|
| Background color | Dark canvas color behind the centered player |
| Blur (px) | Gaussian blur on background elements behind the player |
| Brightness (%) | Background brightness — 0 is pitch black, 80 is semi-visible |
| Shadow spread (px) | Dark shadow radiating from player edges |
| Max player width (vw) | Player width limit as % of viewport (100 = fullscreen) |
| Side curtain (vw) | Dark gradient curtains on screen edges (0 = disabled) |
| Hide description & comments | Removes the area below the player |
| Ambient lighting | Samples video frame colors and glows them behind the player |

Panel footer: **Reset defaults** · **Save & Apply** · **Close**

> Exit via the **camera button** (old camera icon). After exiting, an **Enter Cinema Mode** button appears in the player controls for easy re-entry without reloading.

### 🧩 Architecture

| Layer | Implementation |
|-------|----------------|
| Bootstrapping | Injected at `document-start`, self-contained IIFE |
| DOM observation | `MutationObserver` × 7 for SPA navigation tracking |
| Settings storage | `localStorage` (per-origin, no server sync) |
| Theming | 49 CSS custom properties injected into `:root` |
| Localisation | External JSON loaded via `GM_getResourceText` at runtime |
| Codec override | `XMLHttpRequest` intercept on `googlevideo.com` streams |
| Style injection | `<style>` tag appended to `<head>` on each page transition |
| Yandex detection | User-agent sniffing at runtime, cached after first call |

## 🎨 Quick CSS Examples

Paste any of these into the **Style Editor → Custom CSS** tab:

```css
/* Hide ads */
ytd-ad-slot-renderer,
.ytd-promoted-sparkles-web-renderer,
#masthead-ad {
  display: none !important;
}

/* Rounded video thumbnails */
ytd-thumbnail {
  border-radius: 12px !important;
  overflow: hidden !important;
}

/* Hide Shorts from sidebar */
ytd-guide-entry-renderer[title="Shorts"] {
  display: none !important;
}

/* Wider video description */
#description.ytd-watch-metadata {
  max-width: 100% !important;
}

/* Hide end screen cards */
.ytp-ce-element {
  display: none !important;
}

/* Hide sidebar "Навигатор" section (Music, Movies, Gaming) */
ytd-guide-section-renderer:has(a[href="/gaming"]) {
  display: none !important;
}

/* Hide sidebar "Другие возможности" section (Premium, YT Music, YT Kids) */
ytd-guide-section-renderer:has(a[href="/premium"]) {
  display: none !important;
}
```
