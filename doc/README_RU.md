# 📺 YouTube Fix for Yandex

![Баннер](https://github.com/user-attachments/assets/05972b03-8bcd-4eab-a992-6377db5e68ff)

**Мощный пользовательский скрипт, который устраняет баги YouTube, восстанавливает сломанные функции и даёт полный контроль над интерфейсом — с глубокой оптимизацией специально для Яндекс Браузера.**

После установки **⚙️ кнопка-шестерёнка** появляется на верхней панели YouTube — прямо рядом с кнопкой **+Создать** и иконкой профиля.  
Нажмите её — откроется полная панель настроек: без браузерных расширений, без всплывающих окон, просто плавающее окно, встроенное в страницу.

1. Установите менеджер пользовательских скриптов:

   [![Violentmonkey](https://img.shields.io/badge/Violentmonkey-рекомендуется-8957e5?style=for-the-badge&logo=googlechrome&logoColor=white&labelColor=0d1117)](https://violentmonkey.github.io/get-it/)&nbsp;
   [![Tampermonkey](https://img.shields.io/badge/Tampermonkey-установить-f0b429?style=for-the-badge&logo=googlechrome&logoColor=white&labelColor=0d1117)](https://www.tampermonkey.net/)

2. Нажмите кнопку установки скрипта:

   [![Установить последнюю версию](https://img.shields.io/badge/⬇️_Установить-Последняя_версия-238636?style=for-the-badge&logo=javascript&logoColor=white&labelColor=0d1117)](https://raw.githubusercontent.com/Xanixsl/YouTube-Fix-for-Yandex/main/youtube-fix-yandex.user.js)

> Скрипт автоматически обновляется при выходе новых версий.

## ⚙️ Функции

Панель настроек содержит **3 вкладки**:

### 📂 Вкладка 1 — Общее

#### 🧩 Раздел «Интерфейс»

| Параметр | Описание |
|----------|----------|
| **Скрыть чипсы (фильтры)** | Прячет полосу фильтров на главной странице и в разделах. На страницах каналов чипсы сохраняются |
| **Высота фона чипбара (px)** | Регулирует высоту полосы «frosted-glass» под шапкой. `0` — полностью убрать |
| **Скрыть фон чипбара** | Полностью скрывает фон шапки (для не-Яндекс браузеров) |
| **Компактный режим** | Уменьшает отступы между видео для более плотного расположения |
| **Скрыть Shorts** | Убирает раздел Shorts и все рекомендации коротких видео |
| **Скрыть «Ещё темы»** | Убирает тематические подборки видео («Ещё темы») с главной страницы |
| **Скрыть предупреждение о замедлении** | Убирает уведомление о замедлении YouTube в РФ |
| **Скрыть пустые блоки** | Скрывает пустые плейсхолдеры и сломанные промо-блоки в ленте |
| **Фикс карточки канала на вкладках** | Исправляет «съезжающую» карточку канала на всех вкладках |
| **Восстановить чипсы на вкладке Videos** | Гарантирует отображение сортировочных чипсов на странице канала |
| **Плейлисты на каналах** | Возвращает плейлисты на страницы каналов (отключает оптимизацию Яндекса) |

#### 🐛 Раздел «Исправления багов»

| Параметр | Описание |
|----------|----------|
| **Авто-закрытие «Видео приостановлено»** | Автоматически нажимает «Продолжить» при паузе по бездействию |
| **Фикс вспышки тёмной темы** | Устраняет белую вспышку при навигации в тёмной теме |
| **Фикс сетки поиска** | Исправляет сетку видео на странице результатов поиска |
| **Фикс мини-плеера** | Исправляет проблемы наложения z-index мини-плеера |
| **Оптимизация скролла** | Уменьшает подтормаживания при прокрутке ленты |
| **Фикс боковой панели** | Устраняет глитчи боковой панели при навигации |
| **Принудительный кодек H264** | Отключает VP9/AV1 — устраняет зависания и подтормаживания видео |

#### 🌐 Раздел «Язык интерфейса»

- **Автоматически** — определяется по языку браузера
- **Русский** / **English**

---

### � Вкладка 2 — Яндекс-Фиксы *(только для Яндекс Браузера)*

#### 🏷 Раздел «Фиксы Яндекс Браузера»

| Параметр | Описание |
|----------|----------|
| **Фикс SPA-навигации** | Исправляет проблемы с кнопкой «Назад» и переходами между страницами |
| **Фикс переполнения страницы** | Устраняет двойной скролл, вызванный оптимизациями Яндекса |
| **Фикс полноэкранного режима** | Устраняет артефакты панелей при просмотре видео в полном экране |
| **Фикс управления плеером** | Исправляет рендеринг элементов управления видеоплеером |

#### 📐 Раздел «Настройки сетки видео»

| Параметр | Описание |
|----------|----------|
| **Количество видео в строке** | Сколько карточек видео отображается в одной строке на главной странице |
| **Сдвиг Chipbar (px)** | Вертикальный сдвиг полосы чипсов. Отрицательные значения — вверх |
| **Сдвиг блока видео (px)** | Вертикальный сдвиг сетки видео. Компенсирует пробел от оптимизаций Яндекса |

#### 🧪 Раздел «Экспериментальные функции»

> ⚠️ Используйте с осторожностью — могут быть нестабильными

| Параметр | Описание |
|----------|----------|
| **Исправить сетку видео** | Фиксит баг с отображением 3 видео в строке |
| **Режим оптимизации** | Улучшает производительность в Яндекс Браузере |
| **Экспериментальный фикс сдвига** | Альтернативный метод исправления интерфейса |

### 🎨 Редактор стилей (Style Editor)

Встроенный визуальный редактор тем — открывается кнопкой «Открыть редактор стилей».

#### 📑 Вкладка «Пресеты»

- Сохранить пресет с пользовательским именем

- Визуальный выбор цветовой схемы из готовых наборов
- Предпросмотр изменений в реальном времени

|----------|----------|

## 🎨 Примеры CSS

Вставьте в **Редактор стилей → Свой CSS**:

```css
/* Скрыть рекламу */
ytd-ad-slot-renderer,
.ytd-promoted-sparkles-web-renderer,
#masthead-ad {
  display: none !important;
}

/* Скруглить миниатюры видео */
ytd-thumbnail {
  border-radius: 12px !important;
  overflow: hidden !important;
}

/* Скрыть Shorts из боковой панели */
ytd-guide-entry-renderer[title="Shorts"] {
  display: none !important;
}

/* Расширить описание видео */
#description.ytd-watch-metadata {
  max-width: 100% !important;
}

/* Спрятать карточки в конце видео */
.ytp-ce-element {
  display: none !important;
}

/* Убрать раздел «Навигатор» (Музыка, Игры, Кино) из боковой панели */
ytd-guide-section-renderer:has(a[href="/gaming"]) {
  display: none !important;
}

/* Убрать раздел «Другие возможности» (Premium, YT Music, YT Kids) */
ytd-guide-section-renderer:has(a[href="/premium"]) {
  display: none !important;
}
```

### 🧩 Архитектура

| Слой | Реализация |
|------|------------|
| Запуск | Внедряется на `document-start` в формате автономного IIFE |
| Наблюдение DOM | `MutationObserver` × 7 — отслеживает SPA-навигацию |
| Хранение настроек | `localStorage` (поддомен, без серверной синхронизации) |
| Темизация | 49 CSS переменных встраиваются в `:root` |
| Локализация | JSON-файлы загружаются через `GM_getResourceText` во время выполнения |
| Перехват кодека | XHR-интерцепт запросов к `googlevideo.com` |
| Внедрение стилей | Тег `<style>` добавляется в `<head>` при каждом переходе на страницу |
| Определение Yandex | User-agent сниффинг, результат кэшируется после первого вызова |
